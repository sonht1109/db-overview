Part III. Storage and Execution

Chapter 10. Trees, Logs, and Internal Structures

Chapter 10 Summary: What We Learned

Page Topic Takeaway in One Sentence
091 B-tree structure in detail Internal nodes hold keys + child pointers; leaf nodes hold keys + values (or record IDs). Height stays low via high fan-out.
092 Splits, merges, and rebalancing When a node overflows, it splits; when under-full, it merges with a neighbor — keeping the tree balanced automatically.
093 Write-ahead logging basics All changes are logged sequentially to the WAL before they touch the main data files — the WAL is the single source of durability.
094 Checkpoints A checkpoint flushes all dirty pages to disk and truncates the WAL — balancing recovery time against I/O overhead.
095 Buffer pools The buffer pool caches frequently accessed pages in memory; its size and eviction policy (LRU, Clock) dominate performance.
096 Dirty pages and flushing A dirty page has been modified in memory but not yet written to disk; the background writer flushes them gradually.
097 Crash recovery overview On restart, replay the WAL from the last checkpoint to redo committed work and undo uncommitted work.
098 Space amplification Space amplification = (physical storage used) / (logical data size). Wasted space from fragmentation, page overhead, deleted rows.
099 Read amplification and write amplification Read amp = bytes read / bytes requested; write amp = bytes written / bytes of user data. Both degrade performance.
Reveal: How do these ideas connect to the next chapter?
Chapter 10 equipped you with the internal mechanics: B-trees organize data, the WAL guarantees durability, and the buffer pool keeps hot pages in memory. But these structures exist to serve a higher purpose: correctness. Chapter 11 introduces transactions — the mechanism that ensures your writes are atomic, consistent, isolated, and durable (ACID). You will learn why grouping operations into transactions is essential, what happens when you COMMIT or ROLLBACK, and how the WAL you just studied makes the D in ACID possible.
